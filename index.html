The app features body transformation visualization technology, custom made workout plans, and a progress tracker. Guitar Hero w/ Arduino

Guitar Hero Mockup built in Processing with an Arduino powered controller. The idea behind this project was to help people with disabilities by providing a minimalistic controller in the form of a glove. This was greatly inspired by the movie Ready Player One's haptic suit which translates body movements to in-game movement. This project was coded during a 24 hour hackathon in which we won best hardware hack.
